# Minecraft Launcher

Minecraft Launcher 是由 Mojang 官方出品的 Minecraft 启动器，用于在 PC 上访问和管理 Minecraft 游戏。

![Minecraft Launcher](http://r2.oscsmc.baka.ac.cn/57d937e4-a9cf-4c86-a7d9-705ad9f8925a.png)

> 信息更新时间：2026-06-20

| 特性                | 详情                      |
| ----------------- | ----------------------- |
| 认证方式              | Microsoft 账户            |
| 支持的模组加载器          | 不支持                     |
| 支持平台              | Windows / macOS / Linux |

## 下载地址

[Minecraft 官网](https://www.minecraft.net/zh-hans/download)。

## 特色功能

- 集中访问 PC 上的 Minecraft 游戏
- 支持下载新的 Minecraft 游戏版本
- 支持访问多人游戏服务器
- 支持管理玩家档案
- 支持从 Microsoft Store 安装或重新安装

## 开源协议

无开源协议。
