# GDLauncher

GDLauncher 是由 GorillaDevs Inc. 维护的跨平台 Minecraft 启动器。当前版本通常被称为 GDLauncher Carbon，使用 Rust 和 SolidJS 重写；旧版 GDLauncher 已停止维护。

![GDLauncher](https://cdn.gdl.gg/assets/website-home-app-screenshot.jpg)

> 信息更新时间：2026-06-19

| 特性                | 详情                                    |
| ----------------- | ------------------------------------- |
| 认证方式              | Microsoft 账户 / 多账户管理                  |
| 支持的模组加载器          | Forge / NeoForge / Fabric / Quilt     |
| 支持平台              | Windows / macOS / Linux               |

## 下载地址

[GDLauncher 官网](https://gdlauncher.com/)。

## 特色功能

- 支持安装 Minecraft 版本、模组加载器、模组和整合包
- 支持 CurseForge、Modrinth 等资源平台
- 内置 Java 管理器，可自动安装合适的 Java 运行时
- 内置自动更新
- 支持多账户管理
- 提供低配置设备模式
- 计划支持从 CurseForge、MultiMC、ATLauncher、Technic、Prism、Modrinth、FTB 等启动器或平台导入 / 导出实例

## 许可证与状态

GDLauncher Carbon 仓库使用 [Business Source License 1.1](https://github.com/gorilla-devs/GDLauncher-Carbon) 发布。旧版 GDLauncher 仓库曾使用 GPL-3.0，但官方已说明旧版已经被重写并停止维护。

GDLauncher 名称和 Logo 为 GorillaDevs Inc. 的商标，使用时需要遵守官方商标要求。
