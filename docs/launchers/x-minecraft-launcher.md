# X Minecraft Launcher

X Minecraft Launcher (XMCL) 是由 Voxelum 维护的跨平台 Minecraft 启动器。官方介绍称它是具有现代用户体验的开源 Minecraft 启动器，并提供节省磁盘空间的 Mod 管理方式。

![X Minecraft Launcher](https://raw.githubusercontent.com/Voxelum/xmcl-page/master/.vitepress/theme/assets/home.png)

> 信息更新时间：2026-06-20

| 特性       | 详情                                |
| -------- | --------------------------------- |
| 认证方式     | Microsoft 账户 / 离线账户 / 第三方皮肤站账户 |
| 支持的模组加载器 | Forge / NeoForge / Fabric / Quilt |
| 支持平台     | Windows / macOS / Linux           |

## 下载地址

[X Minecraft Launcher 官网](https://xmcl.app/)。

[GitHub Releases](https://github.com/Voxelum/x-minecraft-launcher/releases)。

Windows 可通过 `winget install CI010.XMinecraftLauncher` 安装，macOS 可通过官方 Homebrew Tap `voxelum/xmcl` 安装。

## 特色功能

- 支持创建多个实例并隔离版本、模组和启动设置
- 使用硬链接或符号链接管理资源，减少重复文件占用
- 支持在启动器内从 CurseForge、Modrinth 下载资源
- 支持导入和导出 CurseForge、Modrinth 整合包
- 支持自动下载 Minecraft、OptiFine 和 JVM
- 支持用户之间的 P2P 联机
- 支持 Windows 代码签名与 appx、appinstaller 安装方式
