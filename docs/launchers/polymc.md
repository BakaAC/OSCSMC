# PolyMC

PolyMC 是一款跨平台 Minecraft 启动器。它是 MultiMC 的一个分支，并且没有得到 MultiMC 的认可。

![PolyMC](https://raw.githubusercontent.com/PolyMC/polymc.github.io/master/src/img/screenshots/LauncherDark.png)

> 信息更新时间：2026-06-19

| 特性              | 详情                                      |
| ----------------- | ----------------------------------------- |
| 登录方式          | 正版登录 / 离线账户                       |
| 支持的模组加载器   | Forge / Fabric / Quilt / LiteLoader       |
| 跨平台            | Windows / macOS / Linux / Steam Deck      |

## 下载地址

[PolyMC 官网](https://polymc.org/download/)。

## 特色功能

 - 支持管理多个相互隔离的 Minecraft 实例
 - 支持导入和导出实例
 - 支持从 Modrinth、CurseForge、FTB、Technic、ATLauncher 等来源创建实例
 - 支持细粒度 Java 和实例设置
 - 提供 Windows、macOS、Linux 与 Steam Deck 下载入口
 - 翻译项目托管在 Weblate
