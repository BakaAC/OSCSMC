# MultiMC

MultiMC 是知名且历史悠久的跨平台 Minecraft 启动器，重点是稳定、可预测和轻量的多实例管理。

![MultiMC](https://multimc.org/images/screenshots/main.png)

> 信息更新时间：2026-06-20

| 特性                | 详情                                  |
| ----------------- | ----------------------------------- |
| 认证方式              | Microsoft 账户 / 离线账户                 |
| 支持的模组加载器          | Forge / Fabric / LiteLoader / Quilt |
| 支持平台              | Windows / macOS / Linux             |

## 下载地址

[MultiMC 官网](https://multimc.org/#Download)。

## 特色功能

- 支持多个相互隔离的游戏实例
- 支持为每个实例单独覆盖 Java、内存和窗口等设置
- 内置带颜色标记的游戏日志，便于排查崩溃和启动问题
- 支持导入和导出实例
- 支持导入 Modrinth、CurseForge、FTB、Technic、ATLauncher 等来源的整合包
- 对旧版本 Minecraft 有较好的兼容支持
