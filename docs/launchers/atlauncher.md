# ATLauncher

ATLauncher 是一款功能强大的 Minecraft 启动器，其核心特色是整合了多个主流模组包平台（如 CurseForge、Modrinth、Technic 和 FTB），允许用户快速浏览、下载并安装各种模组包。它旨在提供简单且高效的实例管理体验。

![ATLauncher Logo](https://atlauncher.com/assets/img/logo.png)

> 信息更新时间：2026-06-20

| 特性       | 详情                                |
| -------- | --------------------------------- |
| 认证方式     | Microsoft 账户 / 离线账户               |
| 支持的模组加载器 | Forge / NeoForge / Fabric / Quilt |
| 支持平台     | Windows / macOS / Linux           |

## 下载地址

[官方网站](https://atlauncher.com/downloads) / [GitHub Releases](https://github.com/ATLauncher/ATLauncher/releases)。

通用版本也可以下载 [Jar 包](https://download.nodecdn.net/containers/atl/ATLauncher.jar)。

## 特色功能

- 原生支持导入 CurseForge、Modrinth、Technic 和 FTB 的模组包
- 支持直接在启动器内搜索并添加模组
- 内置多种主题，并支持导入 ZIP 格式自定义主题
- 支持直接将 IntelliJ 的 theme.json 文件转换为启动器主题
- 游戏实例拥有独立的设置、模组和版本控制
- 支持多账户切换，提供本地服务器模式辅助 Microsoft 登录
- 启动器自带自动检测更新功能
