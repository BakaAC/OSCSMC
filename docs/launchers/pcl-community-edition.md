# PCL Community Edition

PCL Community Edition (PCL-CE) 是由 [PCL Community](https://github.com/PCL-Community) 维护的 PCL 社区版，基于 PCL 开源代码二次开发，包含主线暂未制作的功能和改进。

![PCL Community Edition](https://www.pclc.cc/img/pcl-ce/Homepage.png)

> 信息更新时间：2026-06-20

| 特性                | 详情                                             |
| ----------------- | ---------------------------------------------- |
| 认证方式              | Microsoft 账户 / 离线账户 / 第三方皮肤站账户                  |
| 支持的模组加载器          | Forge / NeoForge / Fabric / LiteLoader / Quilt |
| 支持平台              | Windows                                        |

## 特色功能

- 基于 PCL 开源代码的社区二次开发版，版本号与主线 PCL 不严格对应
- 提供主线暂未制作的功能和改进
- 熟悉的启动界面
- 资源搜索与下载页面
- 实例管理页面
- 提供稳定版与测试版 Release 通道
- Release 资产附带 GPG 签名文件
