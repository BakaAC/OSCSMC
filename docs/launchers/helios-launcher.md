# Helios Launcher

Helios Launcher（原 Electron Launcher）是由 Daniel Scalzi 维护的 Electron / Node.js 自定义 Minecraft 启动器，面向需要让玩家加入整合服务器，并由启动器处理 Java、Forge 与其他 Mod 安装的场景。

![Helios Launcher](https://i.imgur.com/6o7SmH6.png)

> 信息更新时间：2026-06-20

| 特性                | 详情                          |
| ----------------- | --------------------------- |
| 认证方式              | Microsoft 账户                |
| 支持的模组加载器          | Forge / Fabric / LiteLoader |
| 支持平台              | Windows / macOS / Linux     |

## 特色功能

- 面向自定义整合服务器，可在多个服务器配置间切换并显示所选服务器玩家人数
- 自动验证并更新客户端文件，损坏或不匹配的文件会重新下载
- 自动验证 Java，必要时安装合适的 Java 版本
- 内置新闻源
- 提供设置页和 Java 控制面板
- 支持启动器自更新
- 可查看 Mojang 服务状态
