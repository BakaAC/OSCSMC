# PCL2 (Plain Craft Launcher 2)

PCL2 (Plain Craft Launcher 2) 是由 [龙腾猫跃](https://afdian.net/a/LTCat) 开发的 Minecraft 启动器。PCL 通常用于指代 PCL2，旧版 PCL 已过时。

![Plain Craft Launcher 2](http://r2.oscsmc.baka.ac.cn/c80cb96e-cbdb-4513-bc1e-4e935e15f9ff.png)

> 信息更新时间：2026-06-20

| 特性                | 详情                                             |
| ----------------- | ---------------------------------------------- |
| 认证方式              | Microsoft 账户 / 离线账户 / 第三方皮肤站账户                  |
| 支持的模组加载器          | Forge / NeoForge / Fabric / LiteLoader / Quilt |
| 支持平台              | Windows                                        |

## 下载地址

[PCL 下载](https://meloong.com/afd/p/0164034c016c11ebafcb52540025c377) / [GitHub Releases](https://github.com/Meloong-Git/PCL/releases)。

## 特色功能

- 多线程下载安装
- 支持一键安装模组加载器
- 支持启动器内搜索模组或整合包
- 高度个性化
- 智能崩溃分析
- 内置崩溃库

## 开源协议

PCL 使用 [PCL 分发有限许可](https://github.com/Meloong-Git/PCL/blob/main/LICENCE)，并不是完全开源软件。官方仓库公开了 PCL 的大部分源代码，代码会在每次 PCL 发布更新时同步。
