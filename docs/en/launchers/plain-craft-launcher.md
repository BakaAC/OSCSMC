# PCL2 (Plain Craft Launcher 2)

PCL2 (Plain Craft Launcher 2) is a Minecraft launcher developed by [龙腾猫跃](https://afdian.net/a/LTCat). PCL commonly refers to PCL2, while the old PCL version is outdated.

![Plain Craft Launcher 2](http://r2.oscsmc.baka.ac.cn/c80cb96e-cbdb-4513-bc1e-4e935e15f9ff.png)

> Last verified: 2026-06-20

| Feature               | Details                                                        |
| --------------------- | -------------------------------------------------------------- |
| Authentication        | Microsoft Account / Offline Account / Third-Party Skin Account |
| Supported Mod Loaders | Forge / NeoForge / Fabric / LiteLoader / Quilt                 |
| Platforms             | Windows                                                        |

## Features

- Multi-threaded download and installation
- One-click mod loader installation
- In-launcher search for mods or modpacks
- Highly customizable
- Intelligent crash analysis
- Built-in crash library

## Open Source License

PCL uses the [PCL limited distribution license](https://github.com/Meloong-Git/PCL/blob/main/LICENCE) and is not fully open source. The official repository publishes most of PCL's source code and syncs it each time PCL is released.
