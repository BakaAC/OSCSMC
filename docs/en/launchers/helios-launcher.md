# Helios Launcher

Helios Launcher, formerly Electron Launcher, is a custom Minecraft launcher maintained by Daniel Scalzi and built with Electron and Node.js. It is aimed at joining modded servers while the launcher handles Java, Forge, and other mod installation.

![Helios Launcher](https://i.imgur.com/6o7SmH6.png)

> Last verified: 2026-06-20

| Feature               | Details                     |
| --------------------- | --------------------------- |
| Authentication        | Microsoft Account           |
| Supported Mod Loaders | Forge / Fabric / LiteLoader |
| Platforms             | Windows / macOS / Linux     |

## Features

- Built for custom modded servers, with server configuration switching and selected-server player counts
- Validates and updates client files automatically; corrupt or incorrect files are redownloaded
- Validates Java automatically and installs a suitable version when needed
- Built-in news feed
- Settings page with a Java control panel
- Self-updating launcher
- Mojang service status view
