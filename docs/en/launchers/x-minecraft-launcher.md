# X Minecraft Launcher

X Minecraft Launcher (XMCL) is a cross-platform Minecraft launcher maintained by Voxelum. The official description presents it as an open-source Minecraft launcher with modern UX and a disk-efficient way to manage mods.

![X Minecraft Launcher](https://raw.githubusercontent.com/Voxelum/xmcl-page/master/.vitepress/theme/assets/home.png)

> Information updated: 2026-06-20

| Feature               | Details                                        |
| --------------------- | ---------------------------------------------- |
| Authentication        | Microsoft Account / Offline Account / Third-party Skin Site Account |
| Supported Mod Loaders | Forge / NeoForge / Fabric / Quilt              |
| Supported Platforms   | Windows / macOS / Linux                        |

## Downloads

[X Minecraft Launcher Official Website](https://xmcl.app/).

[GitHub Releases](https://github.com/Voxelum/x-minecraft-launcher/releases).

On Windows, it can be installed with `winget install CI010.XMinecraftLauncher`. On macOS, it can be installed through the official Homebrew tap `voxelum/xmcl`.

## Features

- Supports multiple instances with isolated versions, mods, and launch settings
- Uses hard links or symbolic links to manage resources and reduce duplicate files
- Supports downloading resources from CurseForge and Modrinth inside the launcher
- Supports importing and exporting CurseForge and Modrinth modpacks
- Supports automatic downloading of Minecraft, OptiFine, and JVM
- Supports P2P multiplayer between users
- Supports Windows code signing and appx/appinstaller installation
