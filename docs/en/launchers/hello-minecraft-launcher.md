# HMCL (Hello Minecraft! Launcher)

HMCL (Hello Minecraft! Launcher) is an open-source, cross-platform Minecraft launcher developed by [huanghongxun](https://github.com/huanghongxun).

![Hello Minecraft! Launcher](http://r2.oscsmc.baka.ac.cn/b49622e3-b480-4902-8bd8-a56aa5971bd2.png)

> Last verified: 2026-06-20

| Feature               | Details                                                        |
| --------------------- | -------------------------------------------------------------- |
| Authentication        | Microsoft Account / Offline Account / Third-Party Skin Account |
| Supported Mod Loaders | Forge / NeoForge / Fabric / LiteLoader / Quilt                 |
| Platforms             | Windows / macOS / Linux                                        |

## Downloads

Stable and development versions: [Official Website](https://hmcl.huangyuhui.net/download/) / [GitHub Releases](https://github.com/HMCL-dev/HMCL/releases).

## Features

- Mod management
- Automatic game installation
- Modpack creation
- Import Curse and MultiMC modpacks
- Export HMCL and MultiMC modpacks
- UI customization
- Multiplayer help and documentation
