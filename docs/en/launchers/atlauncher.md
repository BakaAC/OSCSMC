# ATLauncher

ATLauncher is a powerful Minecraft launcher whose core feature is the integration of multiple mainstream modpack platforms (such as CurseForge, Modrinth, Technic, and FTB), allowing users to quickly browse, download, and install various modpacks. It aims to provide a simple and efficient instance management experience.

## Features

- Native support for importing modpacks from CurseForge, Modrinth, Technic, and FTB
- Support for searching and adding mods directly within the launcher
- Built-in multiple themes and support for importing custom ZIP themes
- Support for converting IntelliJ theme.json files directly into launcher themes
- Independent settings, mods, and version control for each game instance
- Multi-account switching and local server mode for Microsoft login assistance
- Automatic updates to the latest launcher version
