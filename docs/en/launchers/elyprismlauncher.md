# ElyPrismLauncher

ElyPrismLauncher (also known as PineconeMC) is a fork of Prism Launcher. Its core feature is the built-in native support for Ely.by accounts while maintaining support for Microsoft accounts (MSA). This launcher aims to provide a seamless and powerful Minecraft launch and management experience for players utilizing the Ely.by skin system and authentication services.

> Last verified: 2026-06-20

| Feature               | Details                           |
| --------------------- | --------------------------------- |
| Authentication        | Microsoft Account / Offline Account / Third-Party Skin Account (Ely.by) |
| Supported Mod Loaders | Forge / NeoForge / Fabric / LiteLoader / Rift / Quilt |
| Platforms             | Windows / macOS / Linux           |

## Features

- Native built-in support for Ely.by, automatically loading skins and capes without needing extra plugins
- Inherits Prism Launcher's powerful multi-instance isolation and flexible version management
- Allows direct searching and installation of modpacks and single mods from Modrinth, CurseForge, and FTB
- Provides detailed Java runtime environment configuration and precise allocation of launch arguments
- Built-in screenshot viewer supporting batch management and quick upload for sharing
- Highly active synchronization and feature updates with its upstream version
