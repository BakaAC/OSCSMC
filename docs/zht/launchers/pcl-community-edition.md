# PCL Community Edition

PCL Community Edition (PCL-CE) 是由 [PCL Community](https://github.com/PCL-Community) 維護的 PCL 社群版，基於 PCL 開放原始碼二次開發，包含主線暫未製作的功能和改進。

![PCL Community Edition](https://www.pclc.cc/img/pcl-ce/Homepage.png)

> 資訊更新時間：2026-06-20

| 特性                | 詳情                                             |
| ----------------- | ---------------------------------------------- |
| 認證方式              | Microsoft 帳戶 / 離線帳戶 / 第三方皮膚站帳戶                  |
| 支援的模組載入器          | Forge / NeoForge / Fabric / LiteLoader / Quilt |
| 支援平台              | Windows                                        |

官方 Release 說明還提供 [Mirror 醬高速下載源](https://mirrorchyan.com/zh/projects?rid=PCL2-CE&source=pcl2ce-gh-release)，需要 Mirror 醬 CDK。

## 特色功能

- 基於 PCL 開放原始碼的社群二次開發版，版本號與主線 PCL 不嚴格對應
- 提供主線暫未製作的功能和改進
- 熟悉的啟動界面
- 資源搜尋與下載頁面
- 實例管理頁面
- 提供穩定版與測試版 Release 通道
- Release 資產附帶 GPG 簽名檔案
