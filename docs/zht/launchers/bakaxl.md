# BakaXL

BakaXL 是由 [TT702](https://afdian.net/a/TT702) 主導開發的 Minecraft 啟動器。

BakaXL 共有 4 個大版本，其中 1.X 和 2.X 已經進入 EOL 狀態。

BakaXL 官方路線圖顯示，BakaXL 3.0 目前僅進行安全維護，BakaXL 4.0 仍在開發中，首個公開預覽版計劃於 2026 年夏季推出。

> 資訊更新時間：2026-06-19

| 登入方式          | 離線登入 / 正版登入 / 第三方皮膚站登入           |

## 下載地址

[BakaXL 官網](https://www.bakaxl.com/)。

## 特色功能

 - 支援多個帳戶檔案
 - 高度個人化
 - 停用聊天追蹤與檢舉
