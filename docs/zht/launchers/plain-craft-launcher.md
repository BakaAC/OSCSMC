# PCL2 (Plain Craft Launcher 2)

PCL2 (Plain Craft Launcher 2) 是由 [龍騰貓躍](https://afdian.net/a/LTCat) 開發的 Minecraft 啟動器。PCL 通常用於指稱 PCL2，舊版 PCL 已過時。

![Plain Craft Launcher 2](http://r2.oscsmc.baka.ac.cn/c80cb96e-cbdb-4513-bc1e-4e935e15f9ff.png)

> 資訊更新時間：2026-06-20

| 特性                | 詳情                                             |
| ----------------- | ---------------------------------------------- |
| 認證方式              | Microsoft 帳戶 / 離線帳戶 / 第三方皮膚站帳戶                  |
| 支援的模組載入器          | Forge / NeoForge / Fabric / LiteLoader / Quilt |
| 支援平台              | Windows                                        |

## 下載地址

[PCL 下載](https://meloong.com/afd/p/0164034c016c11ebafcb52540025c377) / [GitHub Releases](https://github.com/Meloong-Git/PCL/releases)。

## 特色功能

- 多執行緒下載與安裝
- 支援一鍵安裝模組載入器
- 支援在啟動器內搜尋模組或整合包
- 高度個人化
- 智慧崩潰分析
- 內建崩潰庫

## 開源協議

PCL 使用 [PCL 分發有限許可](https://github.com/Meloong-Git/PCL/blob/main/LICENCE)，並不是完全開源軟體。官方倉庫公開了 PCL 的大部分原始碼，程式碼會在每次 PCL 發布更新時同步。
