# PCL2 (Plain Craft Launcher 2)

PCL2 (Plain Craft Launcher 2) 是由 [龍騰貓躍](https://afdian.net/a/LTCat) 開發的 Minecraft 啟動器。

![Plain Craft Launcher 2](http://r2.oscsmc.baka.ac.cn/c80cb96e-cbdb-4513-bc1e-4e935e15f9ff.png)

| 特性                | 詳情                                     |
| ----------------- | -------------------------------------- |
| 認證方式              | Microsoft 帳戶 / 離線帳戶 / 第三方皮膚站帳戶          |
| 支援的模組載入器          | Forge / NeoForge / Fabric / LiteLoader |
| 支援平台              | 僅 Windows                              |

## 下載地址

[龍騰貓躍的愛發電](https://afdian.com/p/0164034c016c11ebafcb52540025c377)。

## 特色功能

- 多執行緒下載與安裝
- 支援一鍵安裝模組載入器
- 支援在啟動器內搜尋模組或整合包
- 高度個人化
- 智慧崩潰分析
- 內建崩潰庫
