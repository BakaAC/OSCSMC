# Helios Launcher

Helios Launcher（原 Electron Launcher）是由 Daniel Scalzi 維護的 Electron / Node.js 自訂 Minecraft 啟動器，面向需要讓玩家加入整合伺服器，並由啟動器處理 Java、Forge 與其他 Mod 安裝的場景。

![Helios Launcher](https://i.imgur.com/6o7SmH6.png)

> 資訊更新時間：2026-06-20

| 特性                | 詳情                          |
| ----------------- | --------------------------- |
| 認證方式              | Microsoft 帳戶                |
| 支援的模組載入器          | Forge / Fabric / LiteLoader |
| 支援平台              | Windows / macOS / Linux     |

## 特色功能

- 面向自訂整合伺服器，可在多個伺服器配置間切換並顯示所選伺服器玩家人數
- 自動驗證並更新用戶端檔案，損壞或不符合的檔案會重新下載
- 自動驗證 Java，必要時安裝合適的 Java 版本
- 內建新聞源
- 提供設定頁和 Java 控制面板
- 支援啟動器自更新
- 可檢視 Mojang 服務狀態
