# Minecraft Launcher

Minecraft Launcher 是由 Mojang 官方出品的 Minecraft 啟動器，用於在 PC 上存取和管理 Minecraft 遊戲。

![Minecraft Launcher](http://r2.oscsmc.baka.ac.cn/57d937e4-a9cf-4c86-a7d9-705ad9f8925a.png)

> 資訊更新時間：2026-06-20

| 特性                | 詳情                      |
| ----------------- | ----------------------- |
| 認證方式              | Microsoft 帳戶            |
| 支援的模組載入器          | 不支援                     |
| 支援平台              | Windows / macOS / Linux |

## 下載地址

[Minecraft 官網](https://www.minecraft.net/zh-hans/download)。

## 特色功能

- 集中存取 PC 上的 Minecraft 遊戲
- 支援下載新的 Minecraft 遊戲版本
- 支援存取多人遊戲伺服器
- 支援管理玩家檔案
- 支援從 Microsoft Store 安裝或重新安裝
