# PolyMC

PolyMC 是一款跨平台 Minecraft 啟動器。它是 MultiMC 的一個分支，並且沒有得到 MultiMC 的認可。

![PolyMC](https://raw.githubusercontent.com/PolyMC/polymc.github.io/master/src/img/screenshots/LauncherDark.png)

> 資訊更新時間：2026-06-19

| 特性              | 詳情                                      |
| ----------------- | ----------------------------------------- |
| 登入方式          | 正版登入 / 離線帳戶                       |
| 支援的模組載入器   | Forge / Fabric / Quilt / LiteLoader       |
| 跨平台            | Windows / macOS / Linux / Steam Deck      |

## 下載地址

[PolyMC 官網](https://polymc.org/download/)。

## 特色功能

 - 支援管理多個相互隔離的 Minecraft 實例
 - 支援匯入和匯出實例
 - 支援從 Modrinth、CurseForge、FTB、Technic、ATLauncher 等來源建立實例
 - 支援細粒度 Java 和實例設定
 - 提供 Windows、macOS、Linux 與 Steam Deck 下載入口
 - 翻譯專案託管在 Weblate
