# X Minecraft Launcher

X Minecraft Launcher (XMCL) 是由 Voxelum 維護的跨平台 Minecraft 啟動器。官方介紹稱它是具有現代使用者體驗的開源 Minecraft 啟動器，並提供節省磁碟空間的 Mod 管理方式。

![X Minecraft Launcher](https://raw.githubusercontent.com/Voxelum/xmcl-page/master/.vitepress/theme/assets/home.png)

> 資訊更新時間：2026-06-20

| 特性       | 詳情                                |
| -------- | --------------------------------- |
| 認證方式     | Microsoft 帳戶 / 離線帳戶 / 第三方皮膚站帳戶 |
| 支援的模組載入器 | Forge / NeoForge / Fabric / Quilt |
| 支援平台     | Windows / macOS / Linux           |

## 下載地址

[X Minecraft Launcher 官網](https://xmcl.app/)。

[GitHub Releases](https://github.com/Voxelum/x-minecraft-launcher/releases)。

Windows 可透過 `winget install CI010.XMinecraftLauncher` 安裝，macOS 可透過官方 Homebrew Tap `voxelum/xmcl` 安裝。

## 特色功能

- 支援建立多個實例並隔離版本、模組和啟動設定
- 使用硬連結或符號連結管理資源，減少重複檔案占用
- 支援在啟動器內從 CurseForge、Modrinth 下載資源
- 支援匯入和匯出 CurseForge、Modrinth 整合包
- 支援自動下載 Minecraft、OptiFine 和 JVM
- 支援使用者之間的 P2P 聯機
- 支援 Windows 程式碼簽署與 appx、appinstaller 安裝方式
