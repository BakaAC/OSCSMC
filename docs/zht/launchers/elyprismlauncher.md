# ElyPrismLauncher

ElyPrismLauncher（又稱 PineconeMC）是 Prism Launcher 的一個分支版本，其核心特性是整合了對 Ely.by 帳戶的原生支援，同時保留了對微軟帳戶（MSA）的支援。該啟動器旨在為使用 Ely.by 皮膚系統和認證服務的玩家提供無縫且強大的 Minecraft 啟動與管理體驗。

![ElyPrismLauncher](https://elyprismlauncher.github.io/assets/png/screenshot.png)

> 資訊更新時間：2026-06-20

| 特性       | 詳情                                |
| -------- | --------------------------------- |
| 認證方式     | Microsoft 帳戶 / 離線帳戶 / 第三方皮膚站帳戶 |
| 支援的模組載入器 | Forge / NeoForge / Fabric / LiteLoader / Rift / Quilt |
| 支援平台     | Windows / macOS / Linux           |

## 特色功能

- 內建對 Ely.by 的原生支援，自動載入皮膚和披風，無需額外部署外掛
- 繼承了 Prism Launcher 強大的多實例隔離及靈活的版本管理方式
- 允許在啟動器內直接檢索並安裝 Modrinth、CurseForge 與 FTB 的模組包和單個模組
- 提供詳細的 Java 執行時配置環境並精確分配啟動參數
- 內建截圖檢視器，支援對遊戲截圖進行批次管理與快速上傳分享
- 隨上游版本保持高度活躍的同步與功能更新
