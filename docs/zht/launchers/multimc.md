# MultiMC

MultiMC 是知名且歷史悠久的跨平台 Minecraft 啟動器，重點是穩定、可預測和輕量的多實例管理。

![MultiMC](https://multimc.org/images/screenshots/main.png)

> 資訊更新時間：2026-06-19

| 特性              | 詳情                                      |
| ----------------- | ----------------------------------------- |
| 登入方式          | 正版登入 / 離線帳戶                       |
| 支援的模組載入器   | Forge / Fabric / Quilt / LiteLoader       |
| 跨平台            | Windows / macOS / Linux                   |

## 下載地址

[MultiMC 官網](https://multimc.org/#Download)。

## 特色功能

 - 支援多個相互隔離的遊戲實例
 - 支援為每個實例單獨覆蓋 Java、記憶體和視窗等設定
 - 內建帶顏色標記的遊戲日誌，便於排查崩潰和啟動問題
 - 支援匯入和匯出實例
 - 支援匯入 Modrinth、CurseForge、FTB、Technic、ATLauncher 等來源的整合包
 - 對舊版本 Minecraft 有較好的相容支援
