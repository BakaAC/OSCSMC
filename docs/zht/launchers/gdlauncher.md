# GDLauncher

GDLauncher 是由 GorillaDevs Inc. 維護的跨平台 Minecraft 啟動器。目前版本通常被稱為 GDLauncher Carbon，使用 Rust 和 SolidJS 重寫；舊版 GDLauncher 已停止維護。

![GDLauncher](https://cdn.gdl.gg/assets/website-home-app-screenshot.jpg)

> 資訊更新時間：2026-06-19

| 特性              | 詳情                                  |
| ----------------- | ------------------------------------- |
| 登入方式          | 正版登入 / 多帳戶管理                 |
| 支援的模組載入器   | Forge / NeoForge / Fabric / Quilt     |
| 跨平台            | Windows / macOS / Linux               |

## 下載地址

[GDLauncher 官網](https://gdlauncher.com/)。

## 特色功能

 - 支援安裝 Minecraft 版本、模組載入器、模組和整合包
 - 支援 CurseForge、Modrinth 等資源平台
 - 內建 Java 管理器，可自動安裝合適的 Java 執行環境
 - 內建自動更新
 - 支援多帳戶管理
 - 提供低配置裝置模式
 - 計劃支援從 CurseForge、MultiMC、ATLauncher、Technic、Prism、Modrinth、FTB 等啟動器或平台匯入 / 匯出實例
